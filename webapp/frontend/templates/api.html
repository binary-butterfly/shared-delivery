Sämtliche Daten afu dieser Seite stehen unter einer freien Lizenz und können so optimal weiterverwendet werden.

Endpunkte

Aktuell gibt es drei Endpunkte:

/api/stores/search

Die Suche kann via GET und via POST bedient werden. Die Ausgabe erfolgt in Form einer Liste von Objekten.

/api/stores/geo

Die Geo-Suche kann ebenfalls via GET und via POST bedient werden. Die Ausgabe erfolgt als GeoJSON

Suchparameter

Folgende Suchparameter sind möglich

  • page: Die Seite bei einer paginierten Ausgabe
  • q: Text für eine gewichtete Volltextsuche
  • lat: Ein Längengrad für eine Umkreissuche
  • lon: Ein Breitengrad für eine Umkreissuche
  • radius: Ein Radius in Metern für eine Umkreissuche

/api/store/STOREID

Unter der Einzelabfrage kann ein einzelnes Store-Objekt abgerufen werden. Dies ist besonders nützlich in Verbindung mit der Geo-Suche.
